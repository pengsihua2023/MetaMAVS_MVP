# Surveillance Report Writer Skill

Reports should be clear, cautious, and research-oriented.

Required sections:

- Executive summary
- Sample summary
- QC summary
- Host removal summary
- Viral detection summary
- Taxonomy interpretation
- Abundance trends
- Novel virus candidates
- Risk assessment
- Human review notes
- Recommended follow-up actions
- Reproducibility information

Preferred language:

- "viral signal detected"
- "candidate viral taxon"
- "putative viral contig"
- "requires confirmatory testing"
- "consistent with, but not proof of"

Avoid:

- "confirmed infection"
- "confirmed outbreak"
- "diagnosis"
- "definitive pathogen presence"

High-risk detections should recommend:

- repeat sequencing
- targeted RT-qPCR
- review of read-level evidence
- review of genome coverage
- contamination check
- public-health expert review
